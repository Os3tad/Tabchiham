# نصب

```sh
git clone https://github.com/ErfanOs3tad/tabchi
cd tabchi
chmod +x bot
./bot install
# and send [y] to finish install
```
* * *
## ساخت ربات
```
./bot create
./bot 1
وارد کردن ایدی عددی سودو#
وارد کردن شماره ربات#
```
## ساخت ربات های بیش تر

```sh
cd tabchi
./bot create
./bot number of bot >> شماره رباتی که به شما میدهد
وارد کردن ایدی عددی سودو#
وارد کردن شماره ربات#
```
* * *
## اتولاچ

cd tabchi
